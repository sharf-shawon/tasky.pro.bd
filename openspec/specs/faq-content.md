# Capability: FAQ Content

Information about features, security, and privacy for users.

## User Stories

- As a user, I want to read an FAQ so that I can understand how my data is protected.
- As a user, I want to easily access the FAQ from the main application.

## Functional Requirements

- **FAQ Page**:
    - Provide a standalone `src/faq.html` page.
    - Include sections for:
        - How to use Tasky.
        - Security and Encryption details (local encryption, no server storage).
        - Privacy policy.
- **Navigation**:
    - Add a visible link to the FAQ page in the main application UI (e.g., footer or settings).

## UI/UX

- Consistent theme with the main application (dark mode, modern typography).
- Clear, readable layout for text-heavy content.
